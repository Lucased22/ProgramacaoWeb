A Linguagem HTML

HyperText Markup Language Logo HTML5 Extensão do arquivo .htm, .html MIME text/html Desenvolvido por World Wide Web Consortium Página oficial html.spec.whatwg.org HTML5 (Hypertext Markup Language, versão 5) é uma linguagem de marcação para a World Wide Web e é uma tecnologia chave da Internet, originalmente proposto por Opera Software.[1] É a quinta versão da linguagem HTML.

Alguns dos recursos da linguagem HMTL vistos até este momento:
Recursos Tags associadas Descrição
Links <a> Criar links entre páginas
Listas <ul> e <li> Criar listas de itens ordenados
<ol> e <li> Criar listas de itens não ordenados
Imagens <img> Inserir Imagens
